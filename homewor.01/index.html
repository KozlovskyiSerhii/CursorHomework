Шпаргалка по основним командам Git

Основні команди:

  • git config --global user.name blabla
  • git config --global user.email blabal@bla
  • git init
  • git status
  • git add
  • git commit
  • git log
  • git checkout
  • git branch
  • git push
  • git pull
  • git merge
  • git clone

.gitignore - це файл(в кореневому каталозі репозиторію) в якому вказуються файли, папки які не індексуються(ігноруються) Git

Команди рівня Advanced:

  • git revert
  • git reset
  • git rebase
  • git cherry-pick
  • git stash
